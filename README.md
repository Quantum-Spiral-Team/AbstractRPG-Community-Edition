# AbstractRPG: Community Edition

## About the Project

This is an unofficial, community-maintained fork of the legendary AbstractRPG mod by VivernBar.

Great news: **VivernBar is active again and has officially granted permission** for the continued development of this Community Edition! You can find proof of his approval and our communication in the official Discord server.

- **Non-profit**: We are not seeking any profit. This mod will never be monetized; we consider it a fundamental part of the community's heritage.
- **No Custom Content**: We do not plan to add new custom content to this fork. All changes are strictly tied to the technical foundation, bug fixing, and optimization.

## Our Goals

Our primary mission is to keep the mod afloat and ensure it remains compatible with modern environments. Specifically, we plan to:

- **Fix Bugs & Crashes**: Resolve the massive backlog of known technical issues and compatibility conflicts.
- **Finish Incomplete Features**: Finalize and properly implement features that are currently in the mod but lack full functionality.
- **Refactor the Codebase**: Reorganize the code to make it readable, maintainable, and developer-friendly.
- *Technical Note*: We are currently considering dropping `HookLib` in favor of migrating to `MixinBooter`, though the final decision is pending.

**How to support us**: The best way to help is to report bugs or crashes in the "Issues" tab (English and Russian are both welcome).

## Status & Downloads

- **Stable Releases**: Once a stable, polished version is ready, it will be published in the Releases tab.
- **Experimental**: You can download the latest developmental builds from the **GitHub Actions** tab.
  > **Important:** Please only download builds that have successfully passed the server test! Use these experimental builds at your own risk.
